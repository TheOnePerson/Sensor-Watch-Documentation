void watch_enable_buttons ( )

Enables the external interrupt controller for use with the buttons.

Note
The BTN_ALARM button runs off of an interrupt in the the RTC controller, not the EIC. If your application ONLY makes use of the alarm button, you do not need to call this method; you can save ~5ÂµA by leaving the EIC disabled and only registering a callback for BTN_ALARM.
